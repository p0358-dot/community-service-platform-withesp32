# ESP32-S3 社区AI助手硬件连接指南

## 🔧 所需硬件清单

### 核心组件
- **ESP32-S3开发板** x1 (推荐ESP32-S3-DevKitC-1)
- **麦克风模块** x1 (INMP441, I2S接口)
- **数字功放模块** x1 (MAX98357A)
- **扬声器** x1 (4-8Ω, 3W)
- **OLED显示屏** x1 (128x64, I2C接口, SSD1306驱动, 可选)
- **按键** x1 (用于测试和交互)
- **LED指示灯** x3 (状态指示)

### 连接线材
- **杜邦线** 若干 (公对公、公对母)
- **面包板** x1 (用于原型制作)
- **电阻** 若干 (1KΩ, 10KΩ)
- **跳线帽** x2 (用于短接配置)

## 🔌 详细连接说明

### 1. 麦克风模块连接 (INMP441)
```
ESP32-S3    →    INMP441
GPIO 4      →    WS (数据选择)
GPIO 5      →    SCK (数据时钟)
GPIO 6      →    SD (数据输出)
3.3V        →    VDD (电源正)
GND         →    GND (接地)
            →    L/R (左/右声道短接)
```

### 2. 数字功放模块连接 (MAX98357A)
```
ESP32-S3    →    MAX98357A
GPIO 7      →    DIN (数字信号)
GPIO 15     →    BCLK (位时钟)
GPIO 16     →    LRC (左/右时钟)
3.3V        →    Vin/VCC (电源输入)
GND         →    GND (接地)
            →    SD (关机频道短接)
            →    GAIN (增益和频道短接)
```

### 3. 扬声器连接
```
MAX98357A   →    扬声器
音频+       →    正极 (通常红线)
音频-       →    负极
```

### 4. OLED显示屏连接 (SSD1306, 可选)
```
ESP32-S3    →    OLED
GPIO 41     →    SDA
GPIO 42     →    SCL
3.3V        →    VCC
GND         →    GND
```

### 5. LED指示灯连接
```
ESP32-S3    →    LED + 1KΩ电阻
GPIO 2      →    状态LED (绿色)
GPIO 3      →    语音LED (蓝色)
GPIO 8      →    错误LED (红色)
```

### 6. 测试按键连接
```
ESP32-S3    →    按键
GPIO 0      →    按键一端
3.3V        →    按键另一端 + 10KΩ下拉电阻到GND
```

## 🎯 功能说明

### LED状态指示
- **绿灯 (GPIO 2)**: WiFi连接状态
  - 常亮: WiFi已连接
  - 闪烁: 连接中
  - 熄灭: 连接断开

- **蓝灯 (GPIO 3)**: 语音处理状态
  - 闪烁: 语音识别中
  - 常亮: 语音播放中

- **红灯 (GPIO 8)**: 错误状态
  - 闪烁: 连接错误
  - 常亮: 系统错误

### 按键功能
- **GPIO 0按键**: 
  - 短按: 发送测试消息给AI
  - 长按: 重启设备

### 音频功能
- **I2S麦克风 (INMP441)**: 高质量音频采集
  - 采样率: 16kHz/44.1kHz
  - 位深度: 16位/24位
  - 接口: I2S数字接口

- **I2S功放 (MAX98357A)**: 数字音频播放
  - 功率: 3.2W (4Ω负载)
  - 效率: 85%
  - 接口: I2S数字接口

### 串口通信
- **USB串口**: 用于调试和命令输入
- **I2S音频**: 用于高质量音频输入输出

## ⚡ 电源要求

### ESP32开发板
- **工作电压**: 3.3V
- **USB供电**: 5V (通过板载稳压器)
- **电流消耗**: 
  - 空闲: ~80mA
  - WiFi活动: ~160mA
  - 峰值: ~240mA

### 外设功耗
- **OLED显示屏**: ~20mA
- **LED指示灯**: ~5-10mA/个
- **语音模块**: ~50-100mA
- **总功耗**: ~200-400mA

## 🛠️ 组装步骤

### 步骤1: 准备面包板
1. 将ESP32开发板插入面包板
2. 确保引脚正确对齐

### 步骤2: 连接OLED显示屏
1. 使用杜邦线连接I2C接口
2. 确保电源连接正确

### 步骤3: 连接LED指示灯
1. 每个LED串联1KΩ限流电阻
2. 注意LED正负极

### 步骤4: 连接按键
1. 一端连接GPIO 0
2. 另一端连接3.3V
3. 添加10KΩ下拉电阻到GND

### 步骤5: 连接语音模块 (可选)
1. 使用软串口连接
2. 注意TX/RX交叉连接

## 🔍 故障排除

### 常见问题

#### 1. OLED显示屏不显示
- 检查I2C地址 (通常是0x3C)
- 检查电源连接
- 确认SCL/SDA连接正确

#### 2. WiFi连接失败
- 检查SSID和密码
- 确认信号强度
- 检查网络配置

#### 3. 语音模块无响应
- 检查波特率设置
- 确认TX/RX连接
- 检查电源供应

#### 4. LED不亮
- 检查电阻值
- 确认GPIO配置
- 检查电路连接
